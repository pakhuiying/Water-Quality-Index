Background

Water quality monitoring is one of the most important pillars in water resource management. However, water quality monitoring can be resource intensive, especially for developing countries with limited resources. As such, Water Quality Indexes (WQI) are developed as a convenient tool to summarise general water quality

Usage

  • The following web application allows user to input measured water quality parameters, which subsequently will generate a WQI, regardless of missing data.
  • Users can also input an ID, date, and location for each observation, which will be saved into a database.
  • The observations will be displayed in interactive charts (Chart.js) and map (Leaflet).
  • The user can also download the data in a csv format for further processing.
  • The user can also download the charts in a png format for their perusal in reports etc.

Output

  • WQINON-WEIGHTED (Water quality parameters have equal weights)
  • WQIWEIGHTED (Water quality parameters have unequal weights)
  • WQIADJUSTED (Adjusted for missing values)
  • WQINON-ADJUSTED (Not adjusted for missing value. By default, gives an NA)
